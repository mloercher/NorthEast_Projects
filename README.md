# Northeast Projects, llc

Business page for Northeast Projects, llc -- a Hudson Valley, NY based Passive House design company. The website is mobile responsive for ease of use on the go. 

## Technologies Used


*React.js
*Google Fonts
*React Router
*React Bootstrap
*HTML
*CSS
*JavaScript
*EmailJS
*Framer
